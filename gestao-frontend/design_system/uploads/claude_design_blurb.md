**Community Hub Webapp**

A community management web application with a flat, illustrated visual style. 
Primary green (#4CAF50) background with warm cream (#F3EFE0) card surfaces. 
Line-art icons with light fills, rounded corners (12–20px radius), and a friendly 
yet organized tone. Color palette: greens (#D5E8D4 → #2D5A27), pink accent (#F8BBD0), 
olive borders (#6B8E23). Typography: rounded sans-serif (Nunito or Poppins), bold 
section titles, compact body text. Grid-based card layout with distinct modules 
for each feature area.

# Texto para o campo "Any other notes?"

The visual reference is a webapp template called "Intentional Community Hub" — 
a 3-column card grid where each card represents a feature module (member directory, 
resource booking, events calendar, governance/voting, time bank, shared inventory, 
announcements). Each card has illustrated icons in green/pink tones, bold uppercase 
section titles, and a light internal section color (#D5E8D4) for sub-areas. 
Buttons use rounded pill shapes in #88C9A1. Status tags use #90EE90 (active) 
and #FF80AB (archived). The overall feel is playful, community-oriented, and 
highly scannable.
